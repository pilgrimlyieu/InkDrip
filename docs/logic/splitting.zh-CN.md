# 分段算法

InkDrip 的语义分段器在保持自然阅读边界的前提下，将书籍拆分为适合 RSS 投递的片段。核心实现位于 [`inkdrip-core/src/splitter/semantic.rs`](/inkdrip-core/src/splitter/semantic.rs)。

## 配置项

| 参数     | 配置键                          | 默认值 | 说明             |
| -------- | ------------------------------- | ------ | ---------------- |
| 目标字数 | `defaults.target_segment_words` | 1500   | 理想片段大小     |
| 最大字数 | `defaults.max_segment_words`    | 2000   | 硬性上限         |
| 最小字数 | `defaults.min_segment_words`    | 500    | 尾部片段合并阈值 |

## 四级分段策略

分段器采用级联策略——每一级是前一级产生过大块时的回退方案。

### 第一级：章节边界

若章节字数 ≤ `max_words`，整章作为一个片段，无需进一步分割。

### 第二级：段落边界

超出 `max_words` 的章节，分段器遍历 HTML 块级元素（`<p>`、`<div>`、`<blockquote>`、`<h1>`–`<h6>`、`<ul>`、`<ol>`、`<pre>`、`<table>`、`<section>`、`<article>`、`<main>`）。将段落累积到缓冲区，当当前累计字数比「加上下一段落后」更接近 `target_words` 时，刷新缓冲区为一个片段——即选择与目标字数偏差最小的切分点。这意味着片段可能略微超过 `target_words`，但能产生更均衡的分段结果。

**容器剥离**：若 `<div>` 等块级元素内含嵌套块元素（如 `<div><p>A</p><p>B</p></div>`），外层容器会被「剥离」，内部块元素被提取出来单独处理。这对 EPUB 内容至关重要——章节内容通常被包裹在容器 div 中。

### 第三级：句子边界

当单个段落超过 `max_words` 时，回退到句子级分割。此处同样采用第二级的「更接近目标」启发式——在与 `target_words` 偏差最小的句子边界处刷新。通过扫描终止标点来检测句子：
- **中日韩终止符**：`。！？`
- **拉丁终止符**：`.!?`（排除 `Mr.`、`Dr.`、`e.g.` 等缩写后的句点）
- **闭合标点**：终止符后紧跟的闭合标点（`」』）】〕｝〉》›»\u{201D}` 等）会被吸附到同一句子中，防止引号在片段边界处被截断到下一片段开头。

### 第四级：词边界（硬分割）

对于极长的句子（如无标点的机器生成文本），分段器执行基于字数的硬分割：逐词遍历去除 HTML 标签的文本，在最接近 `target_words` 的位置切割。

## 尾部片段合并

所有章节处理完成后，检查最后一个片段是否低于 `min_words`。若是，则将其合并到前一个片段中——拼接 HTML 内容并重新计算字数。这避免了给读者投递内容过少的「碎片」片段。

## 标题上下文

每个片段携带 `title_context` 字符串，用于 RSS 条目标题：
- 单片段章节：`"第三章"`
- 多片段章节：`"第三章 (1/4)"`、`"第三章 (2/4)"` 等

## 数据流

```
书籍文件  →  解析器 (epub/markdown/txt)  →  Vec<Chapter>
                                                │
                                                ▼
                                          SemanticSplitter
                                                │
                                    ┌───────────┼───────────┐
                                    ▼           ▼           ▼
                                短章节       段落分割     句子分割
                              (单片段)      (L2 级)     (L3 级)
                                    │           │           │
                                    └─────┬─────┘           │
                                          ▼                 ▼
                                     合并尾部片段      词边界硬分割
                                          │              (L4 级)
                                          ▼
                                    Vec<Segment>  →  数据库
```
